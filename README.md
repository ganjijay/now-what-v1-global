# NowWhat 3.1

Brand: NowWhat
Slogan: Know what to do next.

Flow:
1. Intro and real examples
2. Check my document
3. Google sign-in
4. Three free credits for a new user
5. Photo, screenshot, PDF, or text analysis
6. Reply generation and translation

To update the existing project, copy these folders/files into it and replace:
- app
- lib
- package.json

Keep the existing .env.local.
